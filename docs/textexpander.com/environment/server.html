    137.11

    December 21, 2021

    Additional Team-level and reporting options for Growth and Enterprise tier customers.

    New

    • Team-level filtering in organization reports – If your organization has teams set up, you can now filter your reporting to see only the activity for a specific team’s members. (Growth and Enterprise tiers only)
    • Team Admin access to reports – Team admins within an organization can now access reporting about their teams. (Growth and Enterprise tiers only)
    • User Drill-down reports – When viewing an organization’s Snippets report, you can click on a snippet in the table to run a new report showing the specific users who used the snippet during the time period. (Growth and Enterprise tiers only)
    • Snippet Drill-down reports – When viewing an organization’s Users report, you can click on a user in the table to run a new report showing the specific snippets that user expanded during the time period. (Growth and Enterprise tiers only)
    • When a customer creates an organization account, they’ll now receive a welcome email with resources on how to get started with their organization
    • Many views in the web application now contain help screens when the user has not used the feature before

    Improvements

    • Numerous user interface improvements and clean up
    • Better accessibility via keyboard navigation for several controls in the user interface

    Fixes

    • Fixes an issue that could prevent users from signing into the web application in certain situations

    130

    What's New
    Detailed Release Notes

    130.27

    December 16, 2020

    This update significantly refreshes the user experience.

    New

    • Adds sidebar navigation
    • Refreshes icons
    • Lightens elements to improve ease of reading
    • Adds new UI for user management within a team
    • Introduces team admins: A team can (optionally) designate team admins. Team admins can add and remove existing org users and existing org snippet groups to a team. This allows org admins to delegate this responsibility to other org members. Team admins (who are not org admins) will see only the Teams entry for an organization in the sidebar.

    Improvements

    • Snippet groups added to a team now automatically add the team members to the snippet groups’ members list
