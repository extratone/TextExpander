    TextExpander-Enhanced Apps

    TEXTEXPANDER-ENHANCED APPS for iOS
    (74 and counting…)


    For an even better TextExpander experience, try using one of the TextExpander-enhanced apps listed below. Since support for snippet expansion is built into these apps, they can offer a few more features over using just the TextExpander keyboard.

    • Expand formatted text snippets in apps that support it
    • Expand fill-in snippets
      Notice there is a small “Fill-in” next to each app listed below which has support for fill-in snippets. Not all apps have chosen to implement this feature.
    • Expand snippets using any keyboard you want, not just the TextExpander keyboard
    • iPad users can expand snippets with a bluetooth keyboard

    Setup
    Once you have TextExpander and a TextExpander-enhanced app, there are a couples setting adjustments needed to make them work together.

    1. Under TextExpander’s Settings > TextExpander-Enhanced Apps switch on “Share Snippets.”
    2. In the TextExpander-Enhanced app open the settings or preferences and switch on TextExpander.

    You will also see a button for fetching the snippets from TextExpander. You will need to tap that occasionally to update the snippet set in your supporting app.
